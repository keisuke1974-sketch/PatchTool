PatchTool PWA v5.5

GPT受け渡し専用の固定ID方式パッチ適用ツール。GPT貼付け用コピー時は結果欄を更新せず、適用後のみ完成コードを出力します。

1. 元コード / ID付きコード

通常コードを貼って「GPT貼付け用コピー」を押すと、自動でIDを付与してGPT用テキストをコピーします。

DOC_ID: 未作成 現在ファイル: 未指定 行数: 0 文字数: 0 MAX_ID: 0

2. パッチ

GPTから受け取ったID指定パッチを貼ります。先頭の PATCH_TARGET と現在DOC_IDが一致しない場合は適用を止めます。

3. 検証結果

DOC_ID、対象ID、重複、適用可否を確認します。

4. 完成コード

パッチ適用後の完成コードをコピーまたはファイル保存します。内部IDは自動で除去されます。

行数: 0 文字数: 0
